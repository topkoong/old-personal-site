Remote Code Repository

Designed & Implemented a Remote Code repository, local Client, and communication channel in Object Oriented Design class that is responsible for managing source code resources, e.g., files and documents and providing functionality to check-in, check-out, and browse packages. Used a non-relational database, sockets and supported an HTTP like message structure.

  • Technologies Used: C#, the .Net Windows Presentation Foundation framework, and C++\CLI for the graphical part of each Client
